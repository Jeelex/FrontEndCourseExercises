JavaScript Data Types

Data Type Description Reference Link
String A String is plain text and must always be included inside single or double quotes e.g."Hello", otherwise JavaScript will think it's a variable. Click here for more info.
Number There is only one data type representing numbers in JS and it's called Number. Numbers are written without quotes e.g. 54 Click here for more info.
Boolean The Boolean type returns either true or false and thus is used to answer yes or no questions. Click here for more info.
Object Object is the only NON PRIMITIVE data type in Javascript as it can contain more than just a single value. It cn actually store several properties with different values within them. Click here for more info.
null The null data type has only one value and that is "null". It's a unique JS value which means “nothing”, “empty” or “value unknown”. Click here for more info.
undefined undefined reveals that something does not have a value assigned. Even if a variable has been declared it will still show as undefined if it has no value. Click here for more info.
symbol The symbol type is used to create unique identifiers for objects. Click here for more info.
BigInt The BigInt data type has been created to represent integer values larger than (253-1) which the JS "number" data type cannot. Click here for more info.
